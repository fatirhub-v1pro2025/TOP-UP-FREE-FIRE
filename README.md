# VEZR STORE - Top Up Free Fire

Website ini sudah dioptimalkan untuk dapat diakses dan ditemukan melalui Google.

## Optimasi SEO yang Telah Diterapkan

### 1. Meta Tags SEO
- **Title Tag**: Judul yang deskriptif dan mengandung kata kunci utama
- **Meta Description**: Deskripsi lengkap untuk preview di hasil pencarian Google
- **Meta Keywords**: Kata kunci relevan untuk Free Fire dan top up
- **Meta Robots**: Mengizinkan Google untuk mengindeks dan mengikuti link

### 2. Open Graph & Twitter Cards
- Metadata untuk tampilan bagus saat dibagikan di social media
- Meningkatkan visibilitas di Facebook, Twitter, dan platform lainnya

### 3. Structured Data (JSON-LD)
- Schema.org markup untuk toko online
- Membantu Google memahami konten website
- Menampilkan rating dan review di hasil pencarian

### 4. Robots.txt & Sitemap.xml
- `robots.txt`: Memberitahu Google bahwa situs boleh di-crawl
- `sitemap.xml`: Peta situs untuk membantu Google mengindeks halaman

### 5. Semantic HTML
- Penggunaan heading tags (H1, H2) yang proper
- ARIA labels untuk aksesibilitas
- Struktur HTML yang bersih dan valid

## Cara Agar Website Dapat Muncul di Google

### 1. Publish ke GitHub Pages
Pastikan repository ini sudah di-publish sebagai GitHub Pages:
- Buka Settings repository
- Scroll ke "GitHub Pages"
- Set source branch (biasanya main/master)
- Website akan tersedia di: `https://fatirhub-v1pro2025.github.io/TOP-UP-FREE-FIRE/`

### 2. Submit ke Google Search Console
1. Kunjungi [Google Search Console](https://search.google.com/search-console)
2. Tambahkan property dengan URL: `https://fatirhub-v1pro2025.github.io/TOP-UP-FREE-FIRE/`
3. Verifikasi kepemilikan website
4. Submit sitemap.xml: `https://fatirhub-v1pro2025.github.io/TOP-UP-FREE-FIRE/sitemap.xml`

### 3. Request Indexing
Di Google Search Console:
- Masukkan URL halaman di "URL Inspection"
- Klik "Request Indexing"
- Google akan segera meng-crawl dan mengindeks website

### 4. Tunggu Proses Indexing
- Biasanya membutuhkan 1-3 hari
- Cek status di Google Search Console
- Setelah diindeks, website dapat ditemukan di Google
